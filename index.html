Suppression des données utilisateur

Si vous souhaitez supprimer vos données associées à l'application Mazameer, veuillez suivre les étapes suivantes :

  1. Envoyez-nous une demande à l'adresse e-mail suivante : support@mazameer.com.
  2. Indiquez dans votre demande :
    • Votre nom d'utilisateur ou identifiant dans l'application.
    • Une description claire de votre demande (par exemple, "Je souhaite supprimer toutes mes données associées à Mazameer").
  3. Nous traiterons votre demande dans un délai de 30 jours et vous confirmerons la suppression de vos données.

Pour toute question supplémentaire, veuillez consulter notre Politique de confidentialité.
